### **Tic Tac Toe 🎮**  
A simple Tic Tac Toe game built using **HTML, CSS, and JavaScript**.  

## **📌 Features**  
- Two-player mode (X vs O)  
- Responsive UI design  
- "New Game" & "Reset" functionality  
- Winner announcement with a message  
- Draw detection
